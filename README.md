## Установка
```bash
$ git clone git@github.com:z-ev/tms.git && cd tms 
$ cp .env.example .env
$ echo "127.0.0.1 tms.loc www.tms.loc" | sudo tee -a /etc/hosts
$ make up
$ xdg-open http://tms.loc:8089
```
